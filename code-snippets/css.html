Star Wars Opening Crawl

This code is a creative and mathematically precise implementation of the iconic "Star Wars opening crawl" effect, blending modern CSS3 capabilities with dynamic JavaScript calculations.

Here is a breakdown of its strengths:

  1. Immersive 3D Staging (CSS): The CSS sets the stage for a cinematic experience. By using perspective: 350px; and transform-style: preserve-3d on the parent container, it creates a convincing 3D viewport. The high contrast of yellow text on a black background pays faithful homage to the original source material.
  2. Dynamic Geometry (JavaScript): Rather than relying on hard-coded magic numbers, the animateStory() function calculates the animation path based on the user's actual viewport size (window.innerHeight).
    • Trigonometry in Action: The code uses Math.cos(Math.PI / 6) and Math.sin(Math.PI / 6) (corresponding to a 30-degree angle) to calculate exact initial positions. This ensures the text begins "below" the screen and flows upward at the correct perspective angle regardless of the device screen size.
    • Adaptive Duration: The animation speed (duration: storyDiv.clientHeight * 100) scales intelligently based on the length of the text. Longer stories automatically get more time to scroll, ensuring readability is never compromised.
  3. High-Performance Animation: The code utilizes the modern Web Animations API (element.animate). This is generally more performant than older jQuery-style animations because it allows the browser to optimize the rendering pipeline, resulting in a smooth, jank-free 60fps scroll.
  4. Clean and Modular Structure: The styling is encapsulated within a SASS/SCSS-like nested structure, making it readable and easy to maintain. The separation of the container (div#starWarsStory) from the content (figure.story) allows for easy text updates without breaking the complex 3D perspective logic.

JavaScript


animateStory() {
    const storyDiv = document.querySelector("body div#starWarsStory figure.story");
    const yInitTrans = 0.9 * window.innerHeight * Math.cos(Math.PI / 6);
    const zInitTrans = -Math.sin(Math.PI / 6) * window.innerHeight;
    const yFinalTrans = -(2.1 * storyDiv.clientHeight) + 0.5 * window.innerHeight;
    const zFinalTrans = zInitTrans;
    storyDiv.animate([
        { transform: "rotateX(30deg) translate3d(0px, " + yInitTrans.toString() + "px, " + zInitTrans.toString() + "px)" },
        { transform: "rotateX(30deg) translate3d(0px, " + yFinalTrans.toString() + "px, " + zFinalTrans.toString() + "px)" }
    ], {
        duration: storyDiv.clientHeight * 100,
        iterations: 1
    });
}
